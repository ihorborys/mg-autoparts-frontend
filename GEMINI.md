# Контекст проекту MaxGear

- **Що це:** Фронтенд для інтернет-магазину та платформи підбору автозапчастин.
- **Стек технологій:** [React, Vite, Redux Toolkit].
- **Стиль коду:** Компонентний підхід, функціональні компоненти, чистий код, адаптивність під мобільні пристрої.
- **Особливість:** Головний фокус на максимально зручному, швидкому пошуку і замовленні запчастин за артикулами.
- **Перспектива:** Пошук за VIN-кодами, зручна фільтрація брендів, інші модулі для підвищення продаж.
